\# Projeto B2B - Lakehouse Architecture



Projeto prático para construção de um Data Lakehouse utilizando:



\- Python

\- Databricks

\- Delta Lake

\- Arquitetura em múltiplas camadas (Nickel, Bronze, Silver, Gold)



\## Estrutura do Projeto



\- `src/` → Código fonte

\- `landing\_zone/` → Dados brutos

\- `docs/` → Documentação
